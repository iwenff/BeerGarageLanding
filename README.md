# BEER GARAGE — лендинг

Одностраничный лендинг крафтового бара на **Vite + React + GSAP**.
Тёмная индустриальная тема, янтарный акцент, плавные анимации и полностью
адаптивная мобильная вёрстка.

## Запуск

```bash
npm install      # установить зависимости
npm run dev      # дев-сервер (http://localhost:5173)
npm run build    # сборка в папку dist/
npm run preview  # просмотр собранной версии
```

## ⚠️ Перед публикацией — вставьте реальные ссылки

Все ссылки кнопок собраны в одном месте: [`src/data.js`](src/data.js) → объект `LINKS`.
Сейчас стоят заглушки — замените их:

| Ключ           | Куда ведёт                         |
|----------------|------------------------------------|
| `whatsapp`     | WhatsApp руководителя (`https://wa.me/7XXXXXXXXXX`) |
| `max`          | Профиль/чат руководителя в MAX     |
| `telegram`     | Телеграм-канал бара                |
| `vk`           | Группа ВКонтакте                   |
| `reviewYandex` | Страница отзыва на Яндекс          |
| `review2gis`   | Страница отзыва на 2ГИС            |

Там же, в `data.js`, легко править контент: бургеры (`BURGERS`), архив пива
(`BEERS`), ивенты (`EVENTS` — слайдер, можно добавлять новые), залы (`ROOMS`).

## Структура

- `src/sections/` — секции страницы: Hero, Menu (бургеры), Beers («Уже улетели»),
  Events (слайдер ивентов), Interior (залы), Connect (все кнопки связи), Footer.
- `src/components/` — навбар, плавающая мобильная панель CTA, иконки.
- `public/assets/` — все изображения.

## Контент по разделам

- **Меню** — авторские бургеры (фото с подписями).
- **«Уже улетели»** — пример ассортимента крафта, который уже допили.
- **Ивенты** — «Алко-понг»; секция сделана слайдером, новые ивенты добавляются
  в массив `EVENTS`.
- **Атмосфера** — Main Room и VIP Room.
- **Контакты** — главный блок: написать руководителю (WhatsApp / MAX),
  оставить отзыв (Яндекс / 2ГИС), соцсети (Telegram / VK).

Анимации построены на GSAP; уважается системная настройка
`prefers-reduced-motion` (для пользователей, отключивших анимации).
